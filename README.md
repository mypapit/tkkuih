# tkkuih
A Demo App for combingin TKInter with Tensorflow - an app for recognizing Malay Traditional Kuih 


# Requirements
- Tensorflow 1.13+ or 1.14
- Python (obviously)
- TKinter
- Keras
- pillow / PIL
- numpy

or in Anaconda, you can run :
`pip install tensorflow==1.14 tk keras pillow numpy scikit-learn`
